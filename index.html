1. What is difference between flex-box and grid ?

The basic difference between CSS grid layout and CSS flexbox layout is that flexbox was designed for layout in one dimension - either a row or a column. Grid was designed for two-dimensional layout - rows, and columns at the same time.

2. What is the purpose of media query ?

Media queries are a key part of responsive web design, as they allow you to create different layouts depending on the size of the viewport, but they can also be used to detect other things about the environment your site is running on, for example whether the user is using a touchscreen rather than a mouse.

3. What is css box model and how does it work ?

A box in CSS consists of a content area, which is where any text, images, or other HTML elements are displayed. This is optionally surrounded by padding, a border, and a margin, on one or more sides. The box model describes how these elements work together to create a box as displayed by CSS.

It divides elements into layers: content, padding, border, and margin. The content layer holds the actual content, padding creates space around the content, border creates a visible boundary, and margin creates space between elements. By adjusting these layers, we can control the size, spacing, and positioning of elements on a web page.

4. what is semantic tag & why we need this and give me some example ?

Semantic tags are HTML elements that provide meaning and structure to the content within a web page. They convey the purpose or role of the content to both web browsers and developers. Using semantic tags improves the accessibility, maintainability, and search engine optimization (SEO) of a website.

Here are some examples of semantic tags:
1. `header`: Represents the introductory section of a webpage or a section within the page. It typically contains the site logo, navigation, and other introductory content.
2. `nav`: Defines a section that contains navigation links. It is used to create a menu or a list of links that navigate to different pages or sections within a website.
4. `article`: Represents a self-contained composition within a document, such as a blog post, news article, or forum post. It can be independently distributed or syndicated.

Using semantic tags improves the accessibility of a website by providing more meaningful information to assistive technologies like screen readers. Search engines also benefit from semantic tags as they can better understand and index the content of a webpage. Additionally, semantic tags make it easier for developers to understand and maintain the structure of a webpage, enhancing collaboration and code readability.
